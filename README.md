Gaming has moved from PC, to mobile, to the web — and now to **AI**. This list is for the player at the start of the AI-game era: it tracks the games you can play today where AI *is* the experience — NPCs that genuinely converse, stories that write themselves around your choices, and whole worlds generated by neural networks in real time.

**What counts as an AI game here:** a game that contains a live AI element as part of play — an NPC whose dialogue is driven by a large language model, an AI judge, an AI dungeon master, a neural-network-rendered world. **What does not:** ordinary games that merely happened to be *built* with AI tools. Those are "AI-generated games," a different thing — this list is about *playing with* AI, not games made by it.

## Contributing

Contributions are welcome and appreciated! Found a hot new AI game? Please read the [contribution guidelines](CONTRIBUTING.md) first, then open a pull request.

Quality bar: entries must be **playable today** (or a public demo), **actively maintained**, and use AI as a **live, core part of play** — an LLM NPC, an AI judge, an AI narrator, a neural-rendered world. Games that were merely *built* with AI tools ("AI-generated games") are out of scope, and so are development tools — this list is games you play *with* AI.
